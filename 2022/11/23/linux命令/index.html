Linux

排查CPU过高问题

  • top命令：动态显示系统中进程的相关信息，包括进程的内存占用，CPU占用，进程ID
  • 选出CPU占用高的进程id，执行top -H -p pid
  • jstack pid | grep 线程id，通过该命令查看线程的允许状态

关于top命令：

第一行显示系统当前时间，系统运行时间，当前登录的用户数目

第二行显示系统当前的进程数目，总进程数，正在运行的进程数，睡眠的进程数

第三行显示CPU的相关信息

第四行显示物理内存的总量和使用情况 (使用+空闲+缓存 = 总量)

第五行显示交换区总量和使用情况 (使用+空闲+缓存 = 总量)

  • ps命令：查看系统正在运行的进程的情况，包括进程号，CPU运行进程时间，进程运行的命令

  • netstat命令：用来查看linux系统的网络连接状态 (进程号，端口号)，netstat -ntlp

    • -t:tcp，-u:udp，-a:all，-l:listening，-p:进程id和程序名，-r:路由表，-n:显示ip/port

inode的作用

inode是linux文件系统中的一个数据结构，它存储了文件的元数据，如文件名、文件大小、文件权限、文件访问时间、文件所有者等信息。inode相当于一个文件的索引，还存储了文件的物理位置信息，包括文件数据块的地址、大小和位置等。
